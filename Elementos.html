Elementos exclusivos do HTML 5.0

Inclusão do elemento áudio

Com o HTML 5.0 podemos ter reprodução de multimidias, como áudios, sem precisar de plugins externos.

Basta utilizar por exemplo as tags ( audio autoplay="autoplay" controls="controls") e ( source src="endereço da musica" type="audio/mp3").

O autoplay define que o áudio será tocado assim que estiver pronto, o controls mostra os controles.

Inclusão do elemento vídeo

Com o HTML 5.0 podemos ter a reprodução de multimidias, como vídeos, sem precisar de plugins externos.

Basta utilizar por exemplo as tags ( video width="200" height="200" controls="controls") e (source src="endereço do video" type="video/mp4").

Isso define o tamanho que o vídeo terá, que aparecerá seus controles..

Inclusão dos elementos figure e figcaption

Esses elementos unidos permitem que você adicione imagens ao seu site, adicionados de legenda.

Basta utilizar por exemplo as tags (figure img src="endereço da imagem" e figcaption com a legenda que deseja)

Inclusão do elemento dialog

Esse elemento nos permite criar diálogos com caixinhas ou janelas. É uma quebra de um texto arrastado.

Basta utilizar a tag (dialog open).

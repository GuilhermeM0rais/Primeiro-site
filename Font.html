Font

O marcador () "Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML. É importante ressaltar que dentro do contexto do HTML 4.01 o marcador esta em desuso e será obsoletado no HTML 5.

Como então formatar o conteúdo de uma página HTML se a tag será "descontinuada", perguntaria quem nunca se aprofundou no assunto HTML? A marcação inicial , como a marcação final é obrigatória. Com o CSS. O CSS é como se fosse um "guarda-roupa" onde se têm várias "peças" que as tags ou marcadores "vestem" e se apresentam na vitrine que é o navegador. Aquele "guarda-roupa" Pode estar presente em três locais: Dentro da própria tag, onde ela procura primeiro o que "vestir"; Dentro da tag ;
